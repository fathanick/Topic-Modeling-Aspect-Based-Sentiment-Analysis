1. OVERVIEW
================================================================================

This documentation covers the visualization outputs generated from the LLM-based
Aspect-Based Sentiment Analysis (ABSA) of 1,530 Yogyakarta tourism reviews.

The visualizations are designed to:
- Identify dominant keywords and expressions for each tourism aspect
- Show sentiment distribution patterns across aspects
- Reveal relationships between aspects
- Support qualitative discussion in research papers

Data Source: output/absa_llm_groq/absa_llm_results.csv
Statistics Source: output/absa_llm_groq/absa_llm_summary.csv
Output Location: visualizations/absa_llm/
Total Visualizations: 41 files

2. WORD CLOUD VISUALIZATIONS
================================================================================

2.1 Purpose
-----------
Word clouds visualize the most frequent terms used by visitors when discussing
each tourism aspect. Larger words indicate higher frequency, helping identify
common expressions and vocabulary patterns.

2.2 Types of Word Clouds
------------------------
For each of the 12 aspects, up to 3 word clouds are generated:

| Type | Filename Pattern | Description |
|------|------------------|-------------|
| Combined | wordcloud_{aspect}.png | All sentiment evidence combined |
| Positive | wordcloud_{aspect}_positive.png | Positive sentiment evidence only |
| Negative | wordcloud_{aspect}_negative.png | Negative sentiment evidence only |

2.3 Word Cloud Configuration
----------------------------
| Parameter | Value |
|-----------|-------|
| Dimensions | 800 x 400 pixels |
| Max Words | 100 |
| Min Font Size | 10 |
| Max Font Size | 100 |
| Background | White |
| Colormap | Blues (combined), Greens (positive), Reds (negative) |

2.4 Text Preprocessing
----------------------
Before generating word clouds, text is processed:
1. Convert to lowercase
2. Remove special characters (keep alphanumeric and spaces)
3. Tokenize into words
4. Remove stop words (Indonesian + English, 50+ words)
5. Filter words with length <= 2

Stop words filtered: yang, dan, di, ke, dari, untuk, dengan, adalah, ini, itu,
juga, ada, bisa, sangat, sekali, banget, nya, ya, sih, aja, sudah, tidak, ga,
gak, nggak, tapi, atau, kalau, kalo, jadi, buat, sama, lagi, pas, saat, waktu,
disini, kesini, sini, tempat, tempatnya, the, and, to, of, a, in, for, is, on

3. SENTIMENT DISTRIBUTION VISUALIZATIONS
================================================================================

3.1 Sentiment Distribution by Aspect (sentiment_distribution_by_aspect.png)
---------------------------------------------------------------------------
Horizontal stacked bar chart showing the percentage breakdown of positive,
negative, and neutral sentiments for each aspect.

Features:
- Aspects sorted by sentiment score (lowest to highest)
- Color coding: Green (positive), Gray (neutral), Red (negative)
- Sample size (n) annotated for each aspect
- X-axis shows percentage (0-100%)

3.2 Sentiment Scores by Aspect (sentiment_scores_by_aspect.png)
---------------------------------------------------------------
Horizontal bar chart displaying the sentiment score for each aspect.

Sentiment Score Formula: (Positive - Negative) / Total

Score Interpretation:
| Range | Color | Meaning |
|-------|-------|---------|
| 0.6 - 1.0 | Green | Strong positive sentiment |
| 0.3 - 0.6 | Orange | Moderate positive sentiment |
| < 0.3 | Red | Weak/mixed sentiment (needs improvement) |

3.3 Mentions by Aspect (mentions_by_aspect.png)
-----------------------------------------------
Horizontal bar chart showing the number of times each aspect was mentioned
across all reviews. Indicates which aspects visitors discuss most frequently.

4. ASPECT CO-OCCURRENCE ANALYSIS
================================================================================

4.1 Co-occurrence Heatmap (aspect_cooccurrence_heatmap.png)
-----------------------------------------------------------
Matrix visualization showing how often aspects appear together in the same
review. Values are normalized by diagonal (self-occurrence).

Interpretation:
- Value of 1.0 on diagonal = aspect appears with itself (baseline)
- High off-diagonal values = aspects frequently discussed together
- Low values = aspects rarely discussed together

Key Observations:
- Scenery often co-occurs with Atmosphere (visitors describe views + ambiance)
- Facilities co-occurs with Price (visitors evaluate value for money)
- Safety and Crowd often appear together (safety concerns in crowded areas)

4.2 Top Keywords by Aspect (top_keywords_by_aspect.png)
-------------------------------------------------------
Grid of 12 horizontal bar charts showing the top 10 keywords for each aspect.
Useful for identifying the specific vocabulary used when discussing each aspect.

7. USAGE GUIDELINES
================================================================================

7.1 For Research Papers
-----------------------
The visualizations support qualitative discussion paragraphs. Example usage:

"Scenery/View (+0.922) emerged as the highest-rated aspect with 1,032 mentions
and 95.5% positive sentiment. Visitors consistently praised natural beauty using
expressions such as 'pemandangannya indah banget' (the view is very beautiful),
'view sunset mantap' (amazing sunset view), and 'pantainya bagus' (the beach is
nice). The word cloud analysis reveals 'bagus' (good), 'indah' (beautiful), and
'pemandangan' (view) as the dominant vocabulary."

7.2 For Presentations
---------------------
- Use sentiment_scores_by_aspect.png for overall performance comparison
- Use sentiment_distribution_by_aspect.png for detailed breakdown
- Select specific word clouds for aspect-focused discussions

7.3 Regenerating Visualizations
-------------------------------
Run the visualization script:
  python src/visualization/absa_aspect_visualizations.py

Requirements:
- pandas, matplotlib, seaborn, wordcloud
- Input: output/absa_llm_groq/absa_llm_results.csv (evidence text)
- Statistics: output/absa_llm_groq/absa_llm_summary.csv (official counts)
- Output: visualizations/absa_llm/
